==========================
Description of the content:
==========================

in folder Quantum\
##################
1. QLGT_FeynmanRules.nb
--
The propagators and vertices of QLGT. This file should be executed before the one-loop calculations.

2. QLGT_OneLoop_FermionSelfEnergy.nb
--
One-loop correction to the fermion propagator.

3. QLGT_OneLoop_GhostSelfEnergy.nb
--
One-loop correction to the ghost propagator

4. QLGT_OneLoop_VacuumPolarization.nb
--
One-loop correction to the Lorentz gauge propagator

5. QLGT_OneLoop_Vertex.nb
--
One-loop correction to the VAFF vertex

6. V3A.pdf
--
The actual expression for V3A vertex

7. V4A.pdf
--
The actual expression for V4A vertex

8. VAFF.pdf
--
The actual expression for VAFF vertex

9. VAcc.pdf
--
The actual expression for VAcc vertex

in folder Classic\
##################
1. Exact_Solutions_LGT_deSitter.ipynb
--
Python jupyter notebook to prove that de Sitter space is an exact solution of LGT

2. Exact_Solutions_LGT_EarlyUniverse.ipynb
--
Python jupyter notebook to prove that the early universe metric is an exact solution of LGT

3. Exact_Solutions_LGT_MatterDominatedUniverse.ipynb
--
Python jupyter notebook to prove that the matter dominated universe is an exact solution of LGT

4. Exact_Solutions_LGT_Schwarzschild_InMatter.ipynb
--
Python jupyter notebook to prove that Schwarzschild space is an exact solution of LGT in matter

5. Exact_Solutions_LGT_Schwarzschild_Vacuum.ipynb
--
Python jupyter notebook to prove that Schwarzschild space is an exact solution of LGT in vacuum

6.LGT_Kerr.mw:
--
Maple script to show that the Kerr metric is an exact solution of LGT (this file is provided by Marzie Amini and Mehrdad Khodadadifard)

7. LGT_Kerr.pdf:
--
The pdf of file # 6.
